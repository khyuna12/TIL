## **Git 명령어 정리**

<br>

1. `git init` : 로컬 저장소 생성

2. `git add 파일명` : 특정 파일/폴더의 변경사항 추가

3. `git commit -m '커밋메시지'` : 커밋 (버전 기록)

4. `git status` : 상태 확인

5. `git log` : 버전 확인
