## gitignore

- 버전 관리를 별도로 하지 않는 파일/디렉토리가 있을 경우
- Git 저장소에 `.gitignore` 파일 생성하고 관리하기
- 이미 커밋된 파일은 삭제해야 `.gitignore`로 적용됨

<br>

---

<br>

1. 특정 파일 삭제: a.txt(모든 a.txt), test/a.txt(test폴더의 a.txt)

2. 특정 디렉토리: /my_directory

3. 특정 확장자: *.exe

4. 예외 처리: !b.exe
