## CLI
### Command Line Interface

명령줄을 통해서 소통/조작

<br>

### 명령어
<br>

1. ls : list의 약자. 파일 목록을 보여줌

2. touch : 빈 파일을 만듦

3. rm 파일명 : 파일 삭제(remove)

4. mkdir : make directory 폴더를 만듦

5. cd 디렉토리 이름 : change directory

## 버전 관리

소스코드 관리,

## Git
### 분산버전관리시스템(DVCS)으로 코드의 버전을 관리하는 도구

1. 작업을 하고
2. 변경된 파일을 모아(add)
3. 버전으로 남긴다(commit)
